## Overview

A fully custom developer portfolio built to showcase my projects, skills, and experience as an iOS & AI developer. Designed with a dark cinematic aesthetic, fluid animations, and a focus on performance across all devices.

> Built with React 19, TypeScript, Vite, Tailwind CSS, and Framer Motion. Deployed on Vercel.

---

## Features

- **Animated HUD Scanner** — Scan line, biometric equalizer bars, and percentage counter triggered on scroll
- **Magnetic Avatar** — Cursor-following avatar using custom magnet physics
- **Sticky Scroll Project Cards** — Projects revealed in layered sticky scroll
- **Blur Reveal Experience Section** — Role details unblur on hover for a polished effect
- **Marquee Gallery** — Dual-row auto-scrolling GIF reel of project previews
- **Fully Responsive** — Mobile-first design with fluid `clamp()` typography
- **Smooth Section Navigation** — Navbar with smooth scroll to all sections
- **Framer Motion Throughout** — Scroll-triggered, viewport-aware animations everywhere

## Getting Started

```bash
# Clone the repository
git clone https://github.com/NishantSinha7/nishant-portfolio.git

# Navigate into the project
cd nishant-portfolio

# Install dependencies
npm install --legacy-peer-deps

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Deployment

This project is deployed on **Vercel**. Any push to `main` triggers an automatic redeployment.

```bash
# Build for production
npm run build
```
